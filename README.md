# Talkdesk-ai
TalkDeskAI is an AI-powered sales assistant designed to help businesses capture, qualify, and convert website visitors into customers through intelligent conversations.

## 🚀 Features

- Modern and responsive landing page
- AI-powered customer conversations
- Lead capture and qualification
- Fast and intuitive user interface
- Mobile-first responsive design
- Reusable React components
- Built with scalability in mind

## 🛠️ Tech Stack

- React
- Vite
- JavaScript / TypeScript
- Tailwind CSS
- Git & GitHub

```
src/

```

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/talkdeskAI.git
```

### Navigate into the project

```bash
cd talkdeskAI
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
